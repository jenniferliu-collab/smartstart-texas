# SmartStart Texas

Website for SmartStart Texas, an educational consulting business serving Houston families with K–12 school selection, testing prep, and application services.

# Stack

- Single `index.html` file with HTML, CSS, and JS embedded
- Google Fonts: Inter (headings), Roboto (body)
- Contact form through [Formspree](https://formspree.io)
